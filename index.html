XarxatecWordle

Clon del juego "Wordle" como trabajo para clase, utilizando únicamente HTML, Tailwind y Javascript Vanilla.

Instrucciones:

Tienes que adivinar la palabra de 5 letras que el ordenador ha pensado, tendrás que introducir cada letra con el teclado virtual o con el físico. El color verde significa que la letra que has pensado se encuentra en la palabra del ordenador y en el mismo índice. Si sale de color amarillo, significa que la letra está, sin embargo se encuentra en otro índice. Por último si el color es gris , significa que esa letra no se encuentra en la palabra. Tienes un máximo de 6 intentos. Para enviar las letras elegidas, utiliza el enter del teclado virtual o físico.
